🎬 ClipGen - NLW-22 (Rocketseat)
O ClipGen é uma aplicação web desenvolvida durante o evento NLW-22 da Rocketseat. O objetivo do projeto é facilitar a criação de conteúdo curto e viral a partir de vídeos longos enviados pelo usuário, utilizando inteligência artificial para identificar os melhores momentos.

🚀 Tecnologias Utilizadas
O projeto combina ferramentas modernas para garantir performance, design e inteligência:

Gemini API: Inteligência Artificial para análise de conteúdo e identificação de momentos virais.

Cloudinary: Gerenciamento, upload e otimização de vídeos na nuvem.

Tailwind CSS: Estilização responsiva e moderna com foco em utilitários.

GSAP: Animações fluidas e interações de interface de alto nível.

🛠️ Como funciona
O usuário faz o upload de um vídeo longo.

A aplicação utiliza o Cloudinary para processar a mídia.

O Gemini analisa o contexto do vídeo e sugere cortes estratégicos.

O resultado são clipes otimizados prontos para redes sociais como TikTok e Reels.
